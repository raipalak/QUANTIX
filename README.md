📌 Overview

Quantix is a sleek, all-in-one calculator dashboard designed to simplify everyday and academic calculations.
It integrates multiple tools into a single responsive interface, eliminating the need to switch between different apps.

✨ Features

🔥 12 Powerful Modules

Basic & Scientific Calculator

Unit & Currency Converter

Physics & Finance Tools

GPA & Marks Calculator

Matrix Operations

Date Calculator

Geometry & Calculus
